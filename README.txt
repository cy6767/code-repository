CvSU Lost and Found Item Monitoring System
PHP + MySQL Project

HOW TO RUN USING XAMPP:
1. Install XAMPP.
2. Open XAMPP Control Panel.
3. Start Apache and MySQL.
4. Copy the folder cvsu_lostfound to:
   C:\xampp\htdocs\
5. Open browser and go to:
   http://localhost/phpmyadmin
6. Create or import the database using database.sql.
7. Open the project:
   http://localhost/cvsu_lostfound/index.php

PROJECT FEATURES:
- PHP and MySQL connection
- User registration
- Login and logout system
- Session-based access control
- Report lost and found items
- Upload item image
- View lost items
- View found items
- User can edit and delete own reports
- Admin can view, edit, and delete all reports
- Uses prepared statements for database security
- Uses CREATE, INSERT, SELECT, UPDATE, DELETE, WHERE, and JOIN
